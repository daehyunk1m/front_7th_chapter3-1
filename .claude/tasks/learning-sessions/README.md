# 📚 학습 세션 기록

이 폴더에는 Chapter3-1 과제를 진행하면서 기록한 모든 학습 세션이 저장됩니다.

## 📁 파일 구조

```
learning-sessions/
├── README.md                    # 이 파일
├── 2025-11-24-session-1.md     # 2025년 11월 24일 첫 번째 세션
├── 2025-11-24-session-2.md     # 2025년 11월 24일 두 번째 세션
├── 2025-11-25-session-1.md     # 2025년 11월 25일 첫 번째 세션
└── ...
```

## 🎯 목적

- **학습 과정 기록**: 무엇을 배웠는지, 어떤 문제를 해결했는지 상세히 기록
- **PR 작성 도움**: 구체적인 변경 사항과 학습 내용을 PR에 반영
- **회고 자료**: 회고 작성 시 구체적인 근거와 사례 제공
- **성장 추적**: 시간이 지남에 따라 실력 향상 추적

## 📝 세션 기록 방법

### 자동 기록 (권장)

학습을 마칠 때 다음과 같이 말하면 자동으로 기록됩니다:

```
"오늘은 여기까지"
"학습 마무리"
"세션 종료"
```

태스크 매니저가 자동으로:
1. 대화 내용 분석
2. 코드 변경 사항 수집
3. Git 커밋 내역 추출
4. Notion 형식의 마크다운 생성
5. 이 폴더에 저장

### 수동 기록

자동 기록이 작동하지 않을 경우, 다음과 같이 요청할 수 있습니다:

```
"지금까지 학습 내용을 정리해줘"
"오늘 배운 것을 Notion 형식으로 저장해줘"
```

## 📊 세션 파일 형식

각 세션 파일은 다음 섹션을 포함합니다:

### 1. 세션 정보
- 날짜, 시간, 소요 시간
- 진행 Phase, 완료 태스크

### 2. 질문과 답변
- 주요 질문들
- 핵심 답변 요약
- 관련 개념 및 코드

### 3. 학습한 개념
- 새로 배운 기술 개념
- 실습 내용
- 적용 방법

### 4. 작성한 코드
- 주요 파일 변경
- 중요 코드 스니펫
- 코드의 목적과 배운 점

### 5. 문제 해결
- 막혔던 문제들
- 시도한 해결 방법
- 최종 해결책과 교훈

### 6. Git 커밋
- 커밋 내역 테이블
- 변경 파일 목록

### 7. 통계 및 요약
- 세션 통계
- 핵심 학습 요약
- 다음 계획

### 8. PR/회고용 메모
- 한 줄 요약
- PR 작성 시 포함할 내용
- 회고에서 다룰 내용

## 💡 활용 방법

### PR 작성 시

1. 해당 기간의 세션 파일들을 엽니다
2. "작성한 코드" 섹션에서 주요 변경사항 확인
3. "학습한 개념" 섹션에서 기술적 배경 확인
4. "PR/회고용 메모"를 참고하여 PR 설명 작성

### 회고 작성 시

1. 모든 세션 파일의 "핵심 학습 요약" 섹션 확인
2. "문제 해결" 섹션에서 개선 포인트 도출
3. "통계" 섹션으로 정량적 근거 제시
4. "다음 계획"으로 지속적 개선 방향 제시

### 복습 시

1. Phase별로 세션 파일 그룹핑
2. "학습한 개념" 섹션 중심으로 복습
3. "질문과 답변"으로 이해도 재확인
4. "코드" 섹션으로 실습 복기

## 🔍 세션 파일 찾기

### 날짜로 찾기

```bash
# 특정 날짜의 모든 세션
ls 2025-11-24-session-*.md

# 특정 기간
ls 2025-11-2[4-7]-session-*.md
```

### 내용으로 찾기

```bash
# "CVA"가 포함된 세션
grep -l "CVA" *.md

# "Button 컴포넌트"가 포함된 세션
grep -l "Button 컴포넌트" *.md
```

### Phase별로 찾기

```bash
# Phase 1 관련 세션
grep -l "Phase 1" *.md
```

## 🎯 베스트 프랙티스

### 세션 구분 권장

- **2-3시간마다 세션 종료** - 너무 길면 내용이 많아져 정리가 어려움
- **점심/저녁 식사 전후로 구분** - 자연스러운 구분점
- **Phase 전환 시 구분** - Phase별 학습 내용 명확화

### 세션 중 습관

1. **중요한 순간 언급**
   - "이 부분이 핵심이네요"
   - "드디어 이해했어요!"

2. **문제 발생 시 명확히**
   - "이게 안돼요" (구체적으로)
   - "이렇게 해결했어요" (과정 공유)

3. **배운 점 정리**
   - "오늘 CVA를 배웠어요"
   - "TailwindCSS의 장점을 체감했어요"

### 세션 종료 시

1. **간단한 회고**
   - "오늘 뭘 배웠지?"
   - "다음에는 뭘 할까?"

2. **명확한 종료 신호**
   - "오늘은 여기까지"
   - "학습 마무리할게요"
